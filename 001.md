# PythonStock 项目 VPS 安装指南

## 备注：
以下是在一个全新或干净的 VPS （推荐 Debian/Ubuntu 系统）上安装 `pythonstock` 项目的详细步骤。请确保您具有 `sudo` 权限，并在执行任何命令前理解其作用。

---

### 步骤 1：更新系统并安装 Git

首先，更新您的系统软件包列表，并安装 `git`，以便克隆项目。

```bash
sudo apt update
sudo apt install git -y
```

### 步骤 2：安装 Docker

该项目依赖 Docker，您可以使用官方的便捷脚本来安装。

1.  **下载安装脚本：**
    ```bash
    curl -fsSL https://get.docker.com -o get-docker.sh
    ```

2.  **执行脚本以安装 Docker：**
    ```bash
    sudo sh get-docker.sh
    ```

### 步骤 3：安装 Docker Compose

接下来，安装 Docker Compose，它是用于定义和运行多容器 Docker 应用程序的工具。

1.  **下载 Docker Compose：**
    ```bash
    sudo curl -L "https://github.com/docker/compose/releases/download/v2.23.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
    ```

2.  **授予执行权限：**
    ```bash
    sudo chmod +x /usr/local/bin/docker-compose
    ```

### 步骤 4：将您的用户添加到 Docker 组（重要）

为了避免每次使用 `docker` 命令时都输入 `sudo`，请将当前用户添加到 `docker` 用户组。

```bash
sudo usermod -aG docker $USER
```

**注意：** 执行此命令后，您需要 **退出当前的 SSH 会话并重新登录**，这样用户组的更改才会生效。

### 步骤 5：克隆 stock 项目

现在，您可以从 GitHub 克隆 `stock` 项目到您指定的 `stock` 目录中。

```bash
git clone https://github.com/pythonstock/stock.git stock
```

### 步骤 6：启动项目

1.  **进入项目的 `docker-compose` 目录：**
    ```bash
    cd stock/docker-compose
    ```

2.  **使用 Docker Compose 启动服务：**
    为了在 VPS 上稳定运行，建议使用生产环境配置。这个命令会以后台模式构建并启动所有服务（Nginx、后端、数据库等）。
    ```bash
    docker-compose up -d --build
    ```
    启动过程需要一些时间，因为它会下载基础镜像并构建项目。

### 步骤 7：验证和访问

1.  **检查容器是否正在运行：**
    ```bash
    docker ps
    ```
    您应该能看到 `nginx`, `backend`, `mysql-stock` 等几个容器正在运行 (Up)。

2.  **访问项目：**
    项目启动后，您可以通过浏览器访问它。地址是：
    `http://<您的VPS公网IP地址>:8080`

    **防火墙提醒：** 如果无法访问，请确保您的 VPS 的防火墙已经放行了 `8080` 端口。例如，如果您使用 `ufw`，可以执行 `sudo ufw allow 8080`。
